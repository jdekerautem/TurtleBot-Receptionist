# TurtleBot-Receptionist
Implementation of a receptionist based on the turtlebot platform

This project runs on OpenCV 3.0 libraries

ROS Packages (indigo) needed in order to have the nodes run:
  - openni2_launch
  - pocketsphinx
  - sound_play
  - rosserial_server
  - rosserial_arduino
  - rosserial_python
  - uvc_camera
